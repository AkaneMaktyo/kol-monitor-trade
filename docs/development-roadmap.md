# Development Roadmap

更新日期: 2026-05-26

## 近期优先事项

1. 信号决策台
   - 单独展示最新可执行信号、待人工确认信号、仓位更新和最终订单结果
   - 直接接入实时 signal_result，不再只看原始日志

2. 交易就绪面板
   - 明确展示 TRADING_ENABLED、执行模式、交易所凭证状态、风险预算和最近一次下单结果
   - 让用户一眼知道当前是 dry_run 还是可真实提交

3. Gold Empire 解析优先级修正
   - 带 entry 的标准单优先识别为 new_signal
   - 允许先发单、后补 SL/TP
   - `MORE SELL @ price` 既要保留补仓语义，也不能抢掉首单开仓
   - 当前状态: 本地代码已完成首轮修正并通过样例回放验证

4. 回放页
   - 从原消息查看补全文详情、规范化结果、解析结果、风控结果和订单结果
   - 支持按频道/作者回放，降低排障成本

5. 历史检索和定时采样
   - 增加按日期、品种、作者、频道筛选
   - 账户权益曲线改为后台定时采样，不依赖人工刷新页面
